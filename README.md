# 📖 Dictionary App

A web-based Dictionary Application built using HTML, CSS, and JavaScript.  
This application allows users to search for word meanings, definitions, and examples using a Dictionary API.

## 🚀 Features

- Search any English word
- Displays word meaning and definition
- Shows example sentences (if available)
- Simple and clean user interface
- Responsive design

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Dictionary API

## 📂 How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
4. Enter a word in the search box
5. Click the search button to view the meaning

## 🌐 API Used

Free Dictionary API (for fetching word meanings and data)
